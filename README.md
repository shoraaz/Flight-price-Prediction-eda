# Flight Price — Exploratory Data Analysis

> 📦 **Archived learning project.** Kept public for reference; not actively maintained.

## What this is
An exploratory analysis of an Indian domestic flight-fare dataset —
examining how price varies with airline, route, number of stops, departure
time, and days-to-departure.

## Contents
- `flight price eda.ipynb` — the analysis notebook
- `flight_price.xlsx` — the source dataset

## What it demonstrates
- Data cleaning and type handling (dates, durations, categorical fields)
- Feature exploration and correlation analysis
- Visual storytelling with matplotlib / seaborn
